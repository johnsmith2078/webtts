# Web TTS Helper (MVP)

轻量的 Chrome MV3 扩展，在网页选中文本时弹出小按钮，一键用浏览器内置语音朗读。

## 功能（M1 MVP）
- 选中文本后，在选区附近显示悬浮播放按钮。
- 点击按钮使用 `speechSynthesis` 朗读；朗读中按钮变为停止态。
- 朗读结束或停止后自动收起按钮。

## 安装与加载（开发模式）
1. `chrome://extensions` 打开“加载已解压的扩展程序”。
2. 选择本项目根目录（包含 `manifest.json`）。
3. 在任意页面选中一段文字，等待按钮出现并点击播放。

## 手动验证建议
- 选中中英文混合文本，确认自动朗读、停止可用。
- 取消选区后按钮消失；滚动页面后按钮在短暂延迟后消失。
- 朗读过程中再次点击可立即停止。

## 目录
- `manifest.json`：MV3 配置。
- `src/content-script.js`：选区检测、浮动按钮和朗读逻辑。
- `icons/`：占位图标。
- `docs/web-tts-prd.md`：产品文档。
